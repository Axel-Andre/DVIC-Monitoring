# DVIC Monitoring Project

## Installation
Installation du client
```bash
cd app && yarn install
```
Installation du serveur
```bash
cd api && yarn install
```
## Lancement du Project
### Lancer l'API

Dans le dossier [/api](/api), lancez la commande :
```bash
yarn dev
```
### Lancer l'application 

Dans le dossier [/api](/app), lancez la commande :
```bash
yarn dev
```
